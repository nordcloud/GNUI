# Popover

## Popover Props

| properties | required | type            | description                                                   |
| ---------: | :------: | :-------------- | :------------------------------------------------------------ |
|    trigger |   true   | React.ReactNode | trigger element                                               |
|   children |   true   | React.ReactNode | content                                                       |
| alignRight |  false   | boolean         | justifies the content to the right in relation to the trigger |
