# Tooltip

#### Props

> - `caption`: `ReactNode` — **required** (tooltip content)
> - `position`: `left` | `right` — **optional** (default position is center)
> - `bottom`: `boolean` - **optional** (default is false)
> - `status`: `danger` | `warning` | `success` | `notification` — **optional** (background color of tooltip)

## Default Tooltip

> `caption: string;` - caption for the tooltip
