# Toggle

> A toggle button allows changing a setting between two states.

## Toggle properties

| properties | required | type                                                                                       | description                                   |
| ---------: | :------: | :----------------------------------------------------------------------------------------- | :-------------------------------------------- |
|   onChange |   true   | <code>() => boolean</code>                                                                 | React `setState` to handle toggleValue change |
|  labelText |   true   | string                                                                                     |                                               |
|       size |  false   | <code>small</code>                                                                         |                                               |
|     status |  false   | <code>success</code>, <code>danger</code>, <code>warning</code>, <code>notification</code> | color indicator status for SelectButton       |
