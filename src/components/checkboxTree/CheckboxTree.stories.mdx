# CheckboxTree

> The `CheckboxTree` component accepts an array of `Composition` objects and renders it as a tree of checkboxes.

### CheckboxTree properties

|  properties | required |       type        | description                                                         |
| ----------: | :------: | :---------------: | :------------------------------------------------------------------ |
| composition |   true   | CheckboxTreeProps | Composition of how Checkboxes and its childrens should be displayed |
|    onChange |  false   |    () => void     | Returns array with all checked `uid` values                         |
| preSelected |  false   |     string[]      | Array of `uid` values that should be pre selected                   |
|    onExpand |  false   |    () => void     | Returns array with all expanded `uid` values                        |
| preExpanded |  false   |     string[]      | Array of `uid` values that should be pre expanded                   |
