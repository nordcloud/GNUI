# Accordion

> You can build an accordion using a combination of the `Accordion`, `AccordionHeader` and `AccordionItem` components. The Accordion components accepts a list of AccordionHeader components as children and each AccordionHeader accepts a list of AccordionItem is responsible for displaying the content.

## Accordion properties

| properties | required | type                       | description |
| ---------: | :------: | :------------------------- | :---------- |
|   children |   true   | AccordionHeader components |             |

      <AccordionHeader title="Header 2" description="description..." />
    </Accordion>
  </Story>
</Preview>

# AccordionHeader

## AccordionHeader properties

|  properties | required | type                     | description |
| ----------: | :------: | :----------------------- | :---------- |
|       title |   true   | string                   |             |
| description |  false   | string                   |             |
|    children |  false   | AccordionItem components |             |

# AccordionItem

## AccordionItem properties

| properties | required | type                 | description |
| ---------: | :------: | :------------------- | :---------- |
|   children |   true   | string, number, any  |             |
