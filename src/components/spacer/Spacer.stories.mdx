# Spacer

> Used for separate elements

## Spacer properties

| properties | required | type           | description                            |
| ---------: | :------: | :------------- | :------------------------------------- |
|  className |  false   | string         | custom css class name                  |
|     height |  false   | number, string | height of element                      |
|      width |  false   | number, string | width of element                       |
|    bgColor |  false   | string         | background color, default: transparent |
|     border |  false   | string         | border styles                          |
|     margin |  false   | number, string |                                        |
