# List

> Shows list of items

### List properties

|     properties | required | type                                                                                                                                        | description                            |
| -------------: | :------: | :------------------------------------------------------------------------------------------------------------------------------------------ | :------------------------------------- |
|          items |   true   | <code>`Array<{title: string, description: string, rowChildren?: array}>`</code>, <code>`Array<string>`</code>, <code>`Array<number>`</code> |                                        |
|     horizontal |  false   | boolean                                                                                                                                     | default `false`                        |
|       unsorted |  false   | boolean                                                                                                                                     | default `false`                        |
| hasDescription |  false   | boolean                                                                                                                                     | default `false`                        |
|        spacing |  false   | string                                                                                                                                      | used for custom spacing for list items |
