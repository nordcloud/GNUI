# List

> Shows list of items

### List properties

|     properties | required | type                                                                                                                                        | description                            |
| -------------: | :------: | :------------------------------------------------------------------------------------------------------------------------------------------ | :------------------------------------- |
|          items |   true   | <code>`Array<{title: string, description: string, rowChildren?: array}>`</code>, <code>`Array<string>`</code>, <code>`Array<number>`</code> |                                        |
|     horizontal |  false   | boolean                                                                                                                                     | default `false`                        |
|       unsorted |  false   | boolean                                                                                                                                     | default `false`                        |
| hasDescription |  false   | boolean                                                                                                                                     | default `false`                        |
|        spacing |  false   | string                                                                                                                                      | used for custom spacing for list items |

### Default list

> Basic unstyled list. Takes a string or number array

<Canvas>
  <Story name="default">
    <List items={["List item", "Second list item"]} />
  </Story>
</Canvas>

### spacing

> Default list with custom spacing

<Canvas>
  <Story name="spacing">
    <List spacing="0 0 1.15rem 0" items={["List item", "Second list item"]} />
  </Story>
</Canvas>

### horizontal

> Horizontal list. Takes a string or number array

<Canvas>
  <Story name="horizontal">
    <List horizontal items={["List item", "Second list item"]} />
  </Story>
</Canvas>

### unordered

<Canvas>
  <Story name="unordered">
    <List
      unordered
      items={[
        { title: "List item", rowChildren: [{ title: "List item sub row" }] },
        { title: "Second list item" },
      ]}
    />
  </Story>
</Canvas>

### hasDescription

> List with description.

<Canvas>
  <Story name="hasDescription">
    <List
      hasDescription
      items={[
        { title: "List item", description: "List item description" },
        {
