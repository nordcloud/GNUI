# BarChart

> BarChart is used to compare at least two values.

## BarChart properties

| properties | required | type             | description              |
| ---------: | :------: | :--------------- | :----------------------- |
|     values |   true   | `Array<BarItem>` |                          |
|     height |  false   | string           | default is 1.5rem (24px) |

## BarItem properties

| properties | required | type   | description |
| ---------: | :------: | :----- | :---------- |
|      value |   true   | number |             |
|      color |   true   | string |             |
