# PieChart

## PieChart properties

|  properties | required | type                                                                                       | description              |
| ----------: | :------: | :----------------------------------------------------------------------------------------- | :----------------------- |
|        size |   true   | number                                                                                     | width and height in `px` |
|    progress |   true   | number                                                                                     |                          |
| strokeWidth |   true   | number                                                                                     | chart stroke width       |
|       color |  false   | <code>danger</code>, <code>warning</code>, <code>success</code>, <code>notification</code> |                          |
|    children |  false   | <code>`ReactNode`</code>                                                                   | elements inside chart    |
