# BrickLoader

Basing on react-content-loader: https://github.com/danilowoz/react-content-loader

## BrickLoader properties

| properties | required | type   | description                 |
| ---------: | :------: | :----- | :-------------------------- |
|     height |  false   | string | css height value for loader |

As this loader is based on react-content-loader it supports also SVGAttributes and react-content-loader props: https://github.com/danilowoz/react-content-loader#options
