# ListLoader

Basing on react-content-loader: https://github.com/danilowoz/react-content-loader

## ListLoader properties

|  properties | required | type      | description                                       |
| ----------: | :------: | :-------- | :------------------------------------------------ |
| itemsNumber |  false   | number    | number of list rows generated by loader           |
|    children |  false   | ReactNode | custom definition of rows (see format in example) |
