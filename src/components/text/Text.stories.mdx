# Text

> Simple `Text` is styled `p` paragraph.

## Text properties

| properties | required | type                                                                                                  | description                                                                                                                                                 |
| ---------: | :------: | :---------------------------------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------- |
|       size |  false   | <code>xs</code>, <code>sm</code>, <code>md</code>, <code>lg</code>, <code>xl</code>, <code>xxl</code> | Text size                                                                                                                                                   |
|        tag |  false   | string                                                                                                | <code>Text</code> can can be rendered as any other HTML tag instead of p (i.e. <code>Text</code> with <code>tag="span"</code>, will be rendered as a span). |
|    display |  false   | string                                                                                                | used as css <code>display</code> prop                                                                                                                       |
| lineHeight |  false   | string                                                                                                | used as css <code>line-height</code> prop                                                                                                                   |
|      color |  false   | string                                                                                                | used as css <code>color</code> prop                                                                                                                         |
|     weight |  false   | <code>regular</code>, <code>medium</code>, <code>bold</code>                                          | Text weight                                                                                                                                                 |
|  textStyle |  false   | <code>normal</code>, <code>italic</code>                                                              | Text style - italic or normal                                                                                                                               |
|      align |  false   | <code>left</code>, <code>center</code>, <code>right</code>                                            | Text align                                                                                                                                                  |
|      width |  false   | string                                                                                                | used as css <code>width</code> prop                                                                                                                         |
|     nowrap |  false   | boolean                                                                                               |                                                                                                                                                             |
|    isTitle |  false   | boolean                                                                                               |                                                                                                                                                             |

### size

> `size`: `string` — ["xs" | "sm" | "md" | "lg" | "xl" | "xxl"]

<Canvas>
  <Story name="size">
    <GnuiContainer>
      <Text size="xs">(xs) Extra Small size (10px)</Text>
      <Text size="sm">(sm) Small size (12px)</Text>
      <Text size="md">(md) Regular size (16px)</Text>
      <Text size="lg">(lg) Big size (20px)</Text>
      <Text size="xl">(xl) Large size (24px)</Text>
      <Text size="xxl">(xxl) Extra Large size (32px)</Text>
    </GnuiContainer>
  </Story>
</Canvas>

### tag

> `tag`: `string` — `Text` can can be rendered as any other HTML tag instead of `p` (i.e. can be rendered as a `span`). Check this in your dev-tools (<kbd>CTRL</kbd> + <kbd>C</kbd> in Chrome). Uses `as` prop from `styled-components` under the hood.

<Canvas>
  <Story name="tag">
    <GnuiContainer>
      <Text tag="div">
        This is div. <Text tag="span">This is span.</Text>
      </Text>
    </GnuiContainer>
  </Story>
</Canvas>
