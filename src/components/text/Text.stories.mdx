# Text

> Simple `Text` is styled `p` paragraph.

<Preview>
  <Story name="simple">
    <Text>Default paragraph.</Text>
  </Story>
</Preview>

## Props

- `size`: `string` — _optional_
- `tag`: `string` — _optional_
- `display`: `string` — _optional_
- `lineHeight`: `string` — _optional_
- `color`: `boolean` — _optional_
- `weight`: `boolean` — _optional_
- `textStyle`: `boolean` — _optional_
- `align`: `boolean` — _optional_

### size

> `size`: `string` — ["xs" | "sm" | "md" | "lg" | "xl" | "xxl"]

  <Story name="size">
    <GnuiContainer>

### tag

> `tag`: `string` — `Text` can can be rendered as any other HTML tag instead of `p` (i.e. can be rendered as a `span`). Check this in your dev-tools (<kbd>CTRL</kbd> + <kbd>C</kbd> in Chrome). Uses `as` prop from `styled-components` under the hood.

<Preview>
  <Story name="tag">
    <GnuiContainer>
      <Text tag="div">
        This is div. <Text tag="span">This is span.</Text>
      </Text>
    </GnuiContainer>
  </Story>
</Preview>
