# Table

#### Table properties

| properties | required | type    | description                             |
| ---------: | :------: | :------ | :-------------------------------------- |
|    striped |  false   | boolean | table rows will be striped              |
|  hoverline |  false   | boolean | table rows will be highlighted on hover |
|      small |  false   | boolean | table rows will have smaller paddings   |
|       tiny |  false   | boolean | font is smaller                         |
