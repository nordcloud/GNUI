# Table

#### Props

- `striped`: `boolean` — _optional_ - table rows will be striped
- `hoverline`: `boolean` — _optional_ - table rows will be highlighted on hover
- `small`: `boolean` — _optional_ - table rows will have smaller paddings
- `tiny`: `boolean` — _optional_ - font is smaller
