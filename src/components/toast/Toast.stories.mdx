# Toast

## Toast properties

| properties | required | type                                                                                       | description                                 |
| ---------: | :------: | :----------------------------------------------------------------------------------------- | :------------------------------------------ |
|       name |  false   | string                                                                                     |                                             |
|   severity |  false   | <code>danger</code>, <code>notification</code>, <code>warning</code>, <code>success</code> | by default toast has "notification" styling |

## Message properties

| properties | required | type                                       | description                    |
| ---------: | :------: | :----------------------------------------- | :----------------------------- |
|    heading |  false   | string                                     | optional heading for the toast |
|    message |  false   | string                                     | optional message for the toast |
| closeToast |  false   | <code>(e: React.MouseEvent) => void</code> |                                |
