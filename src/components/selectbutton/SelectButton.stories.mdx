# SelectButton

#### Props

- `list`: `array` — **required** and containes the following **required** props: `name`: `string`, `value`: `string`, `labelText`: `string` and the following _optional_: `id`, `className`
- `handleChange`: `() => string | number` — _optional_ - React `setState` to handle select value change
- `value`: `string | number` — _optional_ - current select value
- `size`: `string` — _optional_ - small-sized SelectButton
- `status`: `string` — _optional_ - color indicator status for SelectButton
