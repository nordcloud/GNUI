# Loader

## Loader properties

|   properties | required | type                                                                                                                               | description |
| -----------: | :------: | :--------------------------------------------------------------------------------------------------------------------------------- | :---------- |
|    inContent |  false   | boolean                                                                                                                            |             |
| isBackground |  false   | boolean                                                                                                                            |             |
|     position |  false   | <code>top-left</code> , <code>top-right</code> , <code>bottom-right</code> , <code>bottom-center</code> , <code>bottom-left</code> |             |
