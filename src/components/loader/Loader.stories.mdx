# Loader

## Loader properties

|   properties | required | type                                                                                                                               | description                                                                                                        |
| -----------: | :------: | :--------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------- |
|    inContent |  false   | boolean                                                                                                                            | loader becomes inline instead of fixed, useful for handling loading state inside single components, default: false |
| isBackground |  false   | boolean                                                                                                                            | adds background to fixed loader that handles global loading states, default: false                                 |
|     position |  false   | <code>top-left</code> , <code>top-right</code> , <code>bottom-right</code> , <code>bottom-center</code> , <code>bottom-left</code> |                                                                                                                    |

## Default

> By default Loader position is: top, center
