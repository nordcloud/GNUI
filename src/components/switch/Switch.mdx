# Switch

> A switch allows changing a setting between two states.

## Switch properties

| properties | required | type                                                                                          | description                       |
| ---------: | :------: | :-------------------------------------------------------------------------------------------- | :-------------------------------- |
|  labelText |  false   | string                                                                                        | text to display in the label      |
|   severity |  false   | <code>danger</code> , <code>notification</code> , <code>warning</code> , <code>success</code> | color indicator status for switch |
|   position |  false   | <code>left</code> , <code>right</code>                                                        | switch position                   |
