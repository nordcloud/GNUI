# Page Title with Breadcrumbs

## Page Title with Breadcrumbs properties

| properties | required | type                                  | description |
| ---------: | :------: | :------------------------------------ | :---------- |
|      title |   true   | string                                |             |
|       list |   true   | <code>`Array<BreadcrumbsList>`</code> |             |
|     margin |  false   | string                                |             |

## BreadcrumbsList properties

| properties | required | type   | description |
| ---------: | :------: | :----- | :---------- |
|      label |   true   | string |             |
|        uri |   true   | string |             |
