# Tag

- `color`: `string` – _optional_, background color of Tag by default background is `theme.colors.lights[2]`
- `colorText`: `string` – _optional_, text color of Tag by default text color is `theme.colors.primary`
- `text`: `string` — _optional_, Tag content
- `icon`: `string` — optional, You can use icon prop to add it to the tag. List of available icons you can find [here](?path=/docs/components-svgicon--ui)
- `onClick`: `function` — _optional_
- `showClose`: `boolean` — _optional_, show close button

## Colored Tag with close button

> `color` props can be used in two ways: as an action colors build with `theme` (`accent`, `danger`, `success`, `warning`, `notification`, `primary`) or just in HEX/named color formats according to [X11 Colors](https://en.wikipedia.org/wiki/X11_color_names)
