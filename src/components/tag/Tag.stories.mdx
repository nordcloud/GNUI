# Tag

- `color`: `string` – _optional_, Default Tag background color is `theme.colors.lights[2]` [list of available colors](?path=/docs/foundation-styleguide--colors)
- `colorText`: `string` – _optional_, Default Tag text color is `theme.colors.primary` [list of available colors](?path=/docs/foundation-styleguide--colors)
- `text`: `string` — _optional_, Tag content
- `icon`: `string` — optional, icon to be placed inside the tag [list of available icons](?path=/docs/components-svgicon--ui)
- `onClick`: `function` — _optional_
- `showClose`: `boolean` — _optional_, show close button
- `isTransparent`: `boolean` — _optional_, this prop makes the tag background transparent
