# Tag

*   `color`: `string` – *optional*, Default Tag background color is `theme.colors.lights[2]` [list of available colors](?path=/docs/foundation-styleguide--colors)
*   `colorText`: `string` – *optional*, Default Tag text color is `theme.colors.primary` [list of available colors](?path=/docs/foundation-styleguide--colors)
*   `text`: `string` — *optional*, Tag content
*   `icon`: `string` — optional, icon to be placed inside the tag [list of available icons](?path=/docs/components-svgicon--ui)
*   `onClick`: `function` — *optional*
*   `showClose`: `boolean` — *optional*, show close button
*   `isTransparent`: `boolean` — *optional*, this prop makes the tag background transparent
