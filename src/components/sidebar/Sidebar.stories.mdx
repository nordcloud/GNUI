# Sidebar

> Sidebars by default will be displayed on the right side of the screen. User needs to provide element (button, link, etc) to open it.

## Sidebar properties

|    properties | required | type                                                    | description |
| ------------: | :------: | :------------------------------------------------------ | :---------- |
|      children |  false   | <code>`ReactElement<any>`</code>                        |             |
|         title |  false   | string                                                  |             |
|       caption |  false   | string                                                  |             |
|        isOpen |  false   | boolean                                                 |             |
|          side |  false   | <code>onLeft</code>, <code>onRight</code>               |             |
|         width |  false   | number, string                                          |             |
|       onClick |  false   | <code>(e: any) => void</code>                           |             |
|        footer |  false   | <code>`ReactElement<any>`</code>                        |             |
|  headerStyles |  false   | JSON object containing desired CSS for the header part  |             |
| contentStyles |  false   | JSON object containing desired CSS for the content part |             |

### default

<Canvas>
  <Story name="default">
    {() => {
      const { isOpen, toggle } = useDisclosure(false);
      return (
        <Container>
          <Row justify="center">
            <Button onClick={() => toggle()}>
              Click button to open sidebar
            </Button>
          </Row>
          <Sidebar
            title="Default sidebar"
            caption="Default sidebar caption"
            isOpen={isOpen}
            onClick={() => toggle()}
          >
            Sidebar Content
          </Sidebar>
        </Container>
      );
    }}
  </Story>
</Canvas>

### footer

You can pass Buttons to footer, and it will displayed at the bottom of the Sidebar.

<Canvas>
  <Story name="footer">
    {() => {
      const { isOpen, toggle } = useDisclosure(false);
      const Footer = () => (
        <>
          <Button onClick={() => toggle()}>Cancel</Button>
          <Button>Update</Button>
        </>
      );
      return (
        <Container>
          <Row justify="center">
            <Button onClick={() => toggle()}>
              Click button to open sidebar
            </Button>
          </Row>
          <Sidebar
            title="Default sidebar"
            isOpen={isOpen}
            onClick={() => toggle()}
            footer={<Footer />}
          >
            Sidebar Content
          </Sidebar>
        </Container>
      );
    }}
  </Story>
</Canvas>

### footer with pagination

You can pass PaginationBox, and it will displayed at the bottom of the Sidebar.

<Canvas>
  <Story name="footer pagination">
    {() => {
      const { isOpen, toggle } = useDisclosure(false);
      return (
        <Container>
          <Row justify="center">
            <Button onClick={() => toggle()}>
              Click button to open sidebar
            </Button>
          </Row>
          <Sidebar
            title="Default sidebar"
            isOpen={isOpen}
            onClick={() => toggle()}
            footer={
              <PaginationBox
                sidebar
                small
                count={400}
                from={0}
                size={20}
                setPage={0}
                setSize={20}
              />
            }
          >
            Sidebar Content
          </Sidebar>
        </Container>
      );
    }}
  </Story>
</Canvas>

### side

Sidebar in default is displayed on the right side, but optionally you can change this behaviour with simple prop `side='onLeft'`.

<Canvas>
  <Story name="side">
    {() => {
      const { isOpen: isOpenLeft, toggle: toggleLeft } = useDisclosure(false);
      const { isOpen: isOpenRight, toggle: toggleRight } = useDisclosure(false);
      return (
        <Container>
          <Row justify="center">
            <Text>Click button to open sidebar.</Text>
          </Row>
          <Row justify="space-around">
            <Col md={2}>
              <Button onClick={() => toggleLeft()}>← Left sidebar</Button>
            </Col>
            <Col md={2}>
              <Button onClick={() => toggleRight()}>Right Sidebar →</Button>
            </Col>
          </Row>
          <Sidebar
            title="Default sidebar"
            isOpen={isOpenRight}
            onClick={() => toggleRight()}
          >
            Sidebar Content
          </Sidebar>
          <Sidebar
            side="onLeft"
            title="Left sidebar"
            isOpen={isOpenLeft}
            onClick={() => toggleLeft()}
          >
            Sidebar Content
          </Sidebar>
        </Container>
      );
    }}
  </Story>
</Canvas>

### width

> If you don't need default breakpoints and needs just to hardcode `width` value you can provide it with `width` prop.

<Canvas>
  <Story name="custom width">
    {() => {
      const { isOpen: isOpenPercent, toggle: togglePercent } =
        useDisclosure(false);
      const { isOpen: isOpenRem, toggle: toggleRem } = useDisclosure(false);
      const { isOpen: isOpenPx, toggle: togglePx } = useDisclosure(false);
      return (
        <Container>
          <Row justify="space-around">
            <Col md={2}>
              <Button onClick={() => togglePercent()}>50% Sidebar</Button>
            </Col>
            <Col md={2}>
              <Button onClick={() => toggleRem()}>50rem sidebar on Left</Button>
            </Col>
            <Col md={2}>
              <Button onClick={() => togglePx()}>700px sidebar</Button>
            </Col>
          </Row>
          <Sidebar
            title="Width: 50%"
            width="50%"
            isOpen={isOpenPercent}
            onClick={() => togglePercent()}
          >
            Sidebar Content
          </Sidebar>
          <Sidebar
            title="Width: 50rem"
            width="50rem"
            side="onLeft"
            isOpen={isOpenRem}
            onClick={() => toggleRem()}
          >
            Sidebar Content on Left
          </Sidebar>
          <Sidebar
            title="Width: 700px"
            width="700px"
            isOpen={isOpenPx}
            onClick={() => togglePx()}
          >
            Sidebar Content
          </Sidebar>
        </Container>
      );
    }}
  </Story>
</Canvas>
