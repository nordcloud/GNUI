<Title>Box</Title>

<Description>
  By default `Box` has no borders, margins, paddings nor width and height, but
  all of it (and even more) can be easily styled with `props`. Most of props
  takes values only from `theme`, but not all of them: i.e. `colors` can be
  styled in many ways.
</Description>

## Box properties

|      properties | required | type                                                                                                                                                                                                                                                                                | description                                                                                                     |
| --------------: | :------: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------- |
|            dark |  false   | boolean                                                                                                                                                                                                                                                                             |                                                                                                                 |
|       minHeight |  false   | string                                                                                                                                                                                                                                                                              |                                                                                                                 |
|        minWidth |  false   | string                                                                                                                                                                                                                                                                              |                                                                                                                 |
|          border |  false   | string, or <code>default</code>, <code>medium</code>, <code>bold</code>, <code>grey</code>, <code>disabled</code>, <code>darkenGrey</code>, <code>dark</code>, <code>success</code>, <code>danger</code>, <code>warning</code>, <code>notification</code>, <code>transparent</code> | GNUI as first looks for the value in `theme.colors` and if color doesn't exist there `color` will be used as is |
|           color |  false   | string                                                                                                                                                                                                                                                                              | GNUI as first looks for the value in `theme.colors` and if color doesn't exist there `color` will be used as is |
| backgroundColor |  false   | string                                                                                                                                                                                                                                                                              |                                                                                                                 |
|          radius |  false   | string, or <code>small</code>, <code>default</code>, <code>large</code>                                                                                                                                                                                                             |                                                                                                                 |
|    innerSpacing |  false   | string, or <code>spacing01</code>, <code>spacing02</code>, <code>spacing03</code>, <code>spacing04</code>, <code>spacing05</code>, <code>spacing06</code>, <code>spacing07</code>                                                                                                   |                                                                                                                 |
|         spacing |  false   | string                                                                                                                                                                                                                                                                              |                                                                                                                 |
|          shadow |  false   | string, or <code>shadow01</code>, <code>shadow02</code>, <code>shadow03</code>, <code>shadow04</code>                                                                                                                                                                               |                                                                                                                 |
|        children |  false   | React.ReactNode                                                                                                                                                                                                                                                                     |                                                                                                                 |
|        boxStyle |  false   | <code>lightGrey</code>, <code>grey</code>                                                                                                                                                                                                                                           |                                                                                                                 |

### color

> `color`: `string` — GNUI as first looks for the value in `theme.colors` and if color doesn't exist there `color` will be used as is.
