## Props

```typescript
type BoxProps = {
  dark?: boolean;
  minHeight?: string;
  minWidth?: string;
  border?: string;
  color?: string;
  backgroundColor?: string;
  radius?: string;
  innerSpacing?: string;
  spacing?: string;
  shadow?: string;
  children?: React.ReactNode;
  boxStyle?: "lightGrey" | "grey";
};
```
