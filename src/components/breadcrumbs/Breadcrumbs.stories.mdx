# Breadcrumbs

> The `Breadcrumbs` component accepts a list of `BreadcrumbsList` component and is responsible for displaying the page links in the hierarchy.

## Breadcrumbs properties

| properties | required | type                       | description |
| ---------: | :------: | :------------------------- | :---------- |
|       list |   true   | `Array<BreadcrumbsList>`   |             |
|  Component |  false   | `React.FC<{ to: string }>` |             |

## BreadcrumbsList properties

| properties | required | type    | description |
| ---------: | :------: | :------ | :---------- |
|      label |   true   | string  |             |
|        uri |   true   | string  |             |
| isDisabled |  false   | boolean |             |
