# Toggle

#### Props

- `name`: `string` **required** 
- `labelText`: `string` **required**
- `toggleValue`: `boolean` — _optional_
- `size`: `string` — _optional_ - small-sized SelectButton
- `status`: `string` — _optional_ - color indicator status for SelectButton
- `handleChange`: `() => boolean` — _optional_ - React `setState` to handle toggleValue change

### Default

<Preview>
  <Story name="default">
    {
        () => {
          const [toggle, setToggle] = useState(false);
          return (
              <Toggle labelText="Default" name="default" toggleValue={toggle} handleSelect={setToggle} />
