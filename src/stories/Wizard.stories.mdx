# Wizard

```typescript
interface TabProps {
  className?:string;
  wizard?: boolean;
  step?: number;
  heading?: string;
  caption?:string;
  activeTab: number;
  children?: React.ReactNode;
  onClick?: (e: any) => void;
  props?: any;
  styleActive?:boolean;
  index: number;
  disabled?: boolean;
}

interface TabsProps {
  wizard?: boolean;
  name?: string;
  caption?: string;
  previousButton?:string;
  nextButton?:string;
  submitButton?:string;
  children: Array<TabProps>;
  handleForm: () => void;
}
```

## Default

> `caption?: string;` - caption for the tab
> `previousButton?: string;` - label for Previous button
> `nextButton?: string;` - label for Next button
> `submitButton?: string;` - label for Submit button
