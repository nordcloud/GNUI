# Navigation

#### Props

> By default `Navigation` is styled as primary

- `name`: `string` — _optional_
- `names`: `array` — **required** - array of strings, Navigation's items names
- `links`: `array` — **required** - array of strings, links for Navigation's items
- `onClick`: `(link: string) => void` — **required** - function to handle page chanage 
- `secondary`: `boolean` — _optional_ - secondary navigation
