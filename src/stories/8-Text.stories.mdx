## Props

### as/tag

`Text` can inherit all CSS parameteres, but can be actually rendered as any other HTML tag (e.g. span). Check this in your dev-tools (<kbd>CTRL</kbd> + <kbd>C</kbd> in Chrome).

<Preview>
  <Story name='as/tag'>
    <>
      <Text as='div'>This is div.{" "}
        <Text as='span'>This is span.</Text>
      </Text>
    </>
  </Story>
</Preview>
