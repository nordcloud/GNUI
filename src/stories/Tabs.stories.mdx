# Tabs

#### Props

```typescript
interface TabProps {
  className?: string;
  wizard?: boolean;
  step?: number;
  heading?: string;
  caption?: string;
  activeTab: number;
  children?: React.ReactNode;
  onClick?: (e: any) => void;
  onChange?: (e: any) => void;
  props?: any;
  styleActive?: boolean;
  index: number;
  disabled?: boolean;
}

interface TabsProps {
  wizard?: boolean;
  name?: string;
  caption?: string;
  children: Array<TabProps>;
  handleTab: () => void;
  step: number;
  buttons?: React.ReactNode;
}
```

> `caption?: string;` - caption for the tab

## Wizard With Buttons

> `wizard?: boolean;` - Enable wizard mode

> `buttons?: React.ReactNode;` - Custom buttons on tab

> `buttonsJustify?: string;` - Buttons justify, default: "space-between"
