# Tooltip

#### Props


```typescript
interface TooltipProps {
  name?: string;
  caption?: string;
  position?: "left" | "right" | "top" | "bottom";
  arrowPosition?: "left" | "right";
  status?: "danger" | "warning" | "success" | "notification";
  id?: string;
  className?: string;
  children?: React.ReactNode;
};
```

### Default Tooltip

> `caption?: string;` - caption for the tooltip

### Positions for Tooltip

> - `position?: "left" | "right" | "top" | "bottom";` — position of the tooltip
> - `arrowPosition?: "left" | "right";` — position for the tooltip's arrow, by default arrow is _centered_
