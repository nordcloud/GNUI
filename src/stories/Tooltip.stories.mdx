# Tooltip

#### Props

- `name`: `string` — _optional_
- `caption`: `string` — _optional_ - caption for the tooltip
- `position`: `top, right, bottom, left` — _optional_ - position of the tooltip
- `status`: `danger, warning, success, notification` — _optional_ - table rows will be striped
- `arrowPosition`: `left, right` — _optional_ - position for the tooltip's arrow, by default arrow is _centered_
