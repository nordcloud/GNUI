# Button

## Props

> - `severity`: `medium` | `low` — **optional**
> - `size`: `sm` | `md` — **optional**
> - `icon`: `string` — **optional** you should use icons from this <LinkTo kind="components-svgicon--ui">list of SVG icons</LinkTo>
> - `iconRight`: `boolean` — **optional** if you want to place an icon to the right
> - `initialState`: `success` | `error` | `loading` — **optional**
> - `linkTo`: `string` — **optional**
> - `display`: `flex` | `inline-flex``you might have change Button with linkTo from inline to block` — **optional**

## Button Color

> `color`: `string` — can be used in two ways: as an action colors build with `theme` (`accent`, `danger`, `success`, `warning`, `notification`) or just in HEX/named color formats according to [X11 Colors](https://en.wikipedia.org/wiki/X11_color_names)
