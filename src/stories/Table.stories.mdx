# Table

#### Props

- `name`: `string` — _optional_
- `headings`: `array` — **required** - array of strings, table headings
- `rows`: `array` — **required** - array of arrays of strings, table rows
- `striped`: `boolean` — _optional_ - table rows will be striped
- `hoverline`: `boolean` — _optional_ - table rows will be highlighted on hover
- `small`: `boolean` — _optional_ - table rows will have smaller paddings
- `checkboxes`: `boolean` — _optional_ - table rows will have checkboxes
