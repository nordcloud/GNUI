# Sidebar

<Description>
  Sidebars by default will be displayed on the right side of the screen.
</Description>

<Preview>
  <Story name="default">
    {() => {
      const [isOpen, setIsOpen] = useState(false);
      return (
        <Container>
          <Row justify="center">
            <Button onClick={() => setIsOpen(!isOpen)}>
              Click button to open sidebar
            </Button>
          </Row>
          <Sidebar
            title="Default sidebar"
            isOpen={isOpen}
            onClick={() => setIsOpen(!isOpen)}

    }}
  </Story>

### side

Sidebar in default is displayed on the right side, but optionally you can change this behaviour with simple prop `side='onLeft'`.

<Preview>
  <Story name="side">
    {() => {
      const [isOpen, setIsOpen] = useState(false);
      const [isOpenLeft, setIsOpenLeft] = useState(false);
      return (
        <Container>
          <Row justify="center">
            <Text>Click button to open sidebar.</Text>
          </Row>
          <Row justify="space-around">
            <Col md={2}>
              <Button onClick={() => setIsOpenLeft(!isOpenLeft)}>
                ← Left sidebar
              </Button>
            </Col>
            <Col md={2}>
              <Button onClick={() => setIsOpen(!isOpen)}>
                Right Sidebar →
              </Button>
            </Col>
          </Row>
          <Sidebar
            title="Default sidebar"
            isOpen={isOpen}
            onClick={() => setIsOpen(!isOpen)}
          >
            Sidebar Content
          </Sidebar>
          <Sidebar
            side="onLeft"
            title="Left sidebar"
            isOpen={isOpenLeft}
            onClick={() => setIsOpenLeft(!isOpenLeft)}
          >
            Sidebar Content
          </Sidebar>
        </Container>
      );
    }}
  </Story>
</Preview>
