# Box

By default `Box` has no borders, margins, paddings nor width and height, but all of it (and even more) can be easily styled with `props`. Most of props takes values only from `theme`, but not all of them: i.e. `colors` can be styled in many ways.

<Preview>
  <Story name="box">
    <Box>Default box</Box>
  </Story>
</Preview>

## Props

```typescript
type BoxProps = {
  dark?: boolean;
  minHeight?: string;
  minWidth?: string;
  radius?: string;
  padding?: string;
  spacing?: string;
  shadow?: string;
  border?: string;
  color?: string;
  children?: React.ReactNode;
};
```
