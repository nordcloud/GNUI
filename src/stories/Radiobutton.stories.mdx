> Default input type `radio`.

#### Props

- `name`: `string` — **required**
- `labelText`: `string` — **required** - label for the radio button
- `disabled`: `boolean` — _optional_ - should be styled separately for now
- `required`: `boolean` — _optional_ - should be styled separately for now
