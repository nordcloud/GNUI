# Switch

#### Props

- `name`: `string` — _optional_ - switch name
- `isChecked`: `boolean` — _optional_ - switch value
- `labelText`: `string` — _optional_ - text to display in the label
- `severity`: `"danger" | "notification" | "warning" | "success"` — _optional_ - color indicator status for switch
- `position`: `"left" | "right"` — _optional_ - switch position
- `handleSelect`: `() => boolean` — _optional_ - React `setState` to handle switch change
